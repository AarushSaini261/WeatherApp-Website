# 🌦️ Weather App

A responsive weather application built using HTML, CSS, and JavaScript that provides real-time weather information for cities around the world.

## Features

- Search weather by city name
- Real-time weather data
- Temperature display
- Wind speed information
- Geographic coordinates
- Country and location details
- Clean and responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo API
- Fetch API

## How It Works

1. Enter a city name.
2. The app retrieves location coordinates using the geocoding API.
3. Weather data is fetched from Open-Meteo.
4. Results are displayed instantly.
