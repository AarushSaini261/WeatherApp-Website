# 🌦️ Weather App

A simple and responsive weather application that allows users to search for any city and view real-time weather information including temperature, weather conditions, humidity, wind speed, and location details.

## 🚀 Features

- Search weather by city name
- Real-time weather data
- Displays:
  - Temperature
  - Weather condition
  - Humidity
  - Wind speed
  - Coordinates
  - Country information
- Responsive design for desktop and mobile devices
- Clean and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Weather API
- Fetch API
